# About

## 私について

2025 年に新卒で IT 企業に入社したエンジニアです。

Java / Spring Boot をメインに、AWS も触っています。

設計やアーキテクチャまわりに興味があって、「なぜそう作るのか」を考えるのが好きです。

面白そうなことには大体手を出しています。
&nbsp;

## このブログについて

考えたことや調べたことを、あとで自分が思い出せるくらいの温度で残していきます。

寝たきりでコードが書けない夜でも、ベッドで読み返せるような、そんな深夜のメモみたいなものです。
&nbsp;

## 寝たきり｜Late night について

「寝たきり」は、自分が好きな言葉です。

横になりながらでも考えることはできるし、コーディングできない夜があってもここに知識が積まれていれば意味がある。

・・・そんな意味があったらなと願っています。

&nbsp;

「Late night」は、「寝」から連想されたこと、また私が夜型人間であることから来ています。

そして、このブログ名を「寝たきり｜Late night」という日英混合の形にしたのは、Laura day romance というバンドのオマージュです。

曲名を「日本語｜英語」で並べるスタイルが特徴的で、影響を受けて同じ形にしています。
